#### shrn-cli
cli for our own project(simple version)

##### 构建过程
1、npm init<br>
2、安装commander/inquirer/ora/chalk/download-git-repo(简易版，内容较少)<br>
3、package.json中新增bin,值为bin目录下的index.js<br>
4、npm link<br>
5、index.js中新增实现内容<br>

##### 发布安装
1、npm官网注册账号<br>
2、npm login<br>
3、npm publish<br>
